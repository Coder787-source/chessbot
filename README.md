Hi Everyone,welcome to the repo of ChessBot,an open source project deleveloped entirely in python.Features include downloading games as a .pgn for analysis on chess.com/lichess,stockfish engine integration,multiple levels of diffuculty,time control down to the minute,basic anti-cheat system,custom elo adjustment,error reported in the log of python(ex:CRITCAL ERROR:Stockfish engine not found in [path].


The link to the stockfish executable https://stockfishchess.org/download/
The link to download python is here https://www.python.org/downloads/
Everything else must be downloaded in cmd(windows),Terminal(mac/linux)


Engine integration Credits Goes to the Stcokfish engine authors.
